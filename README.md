## LOTTO PROJECT
> 가상 LOTTO GUI 프로그램  
> 
> 개발기간 : 2024.07.26 ~ 2024.08.06


## What is? 

- 가상 Lotto번호 추첨 및 확인 Windows환경 GUI 프로그램
- 수동, 반자동, 자동 중 원하는 방법으로 로또 번호를 입력 후, 당첨 번호와 비교, 결과확인
